Instalación de Java
en Servidores Web

Aprende a instalar Java EE y otros servicios
en los servidores web más usados

A GUIDE BY MARCO A. BLANCO

  Instalación y configuración en TOMCAT SERVER
Versión en Ubuntu

  1. Descarga de Apache Tomcat:
    Visita el sitio oficial de Apache Tomcat en https://tomcat.apache.org/
    En la sección "Download" elige la versión de Tomcat que desees y descarga el archivo binario (archivo .tar.gz).
  2. Instalación en Ubuntu:
    Abre una terminal. Navega al directorio donde descargaste el archivo Tomcat y descomprime el archivo con el siguiente comando:
    tar -xvzf apache-tomcat-VERSION.tar.gz
    Mueve la carpeta de Tomcat al directorio deseado:
    sudo nano /opt/apache-tomcat-VERSION/bin/catalina.sh
  3. Configuración:
    Abre el archivo de configuración catalina.sh para definir variables de entorno:
    sudo nano /opt/apache-tomcat-VERSION/bin/catalina.sh
    Agrega la línea siguiente al principio del archivo para especificar la ruta de Java:
    export JAVA_HOME=/usr/lib/jvm/openjdk-VERSION
  4. Iniciar Apache Tomcat:
    Desde la terminal, ejecuta el siguiente comando:
    /opt/apache-tomcat-VERSION/bin/startup.sh
    Desde la terminal, ejecuta el siguiente comando:
    /opt/apache-tomcat-VERSION/bin/startup.sh
  5. Abre un navegador web y accede a http://localhost:8080 para el panel de administración de Tomcat. Debes poder ver la página de inicio de Tomcat.
